Antes de mais nada, vamos aos primeiros passos da Tecnologia

Em 1792 Um Francês chamado Claude Chappe , desenvolveu um modo de comunicação chamado Telégrafo Semafórico, através de painéis fixados no alto das torres, as pessoas responsáveis pela própria torre, transmitiam a mensagem para a torre seguinte utilizando uma luneta através de códigos, um método bastante limitado, não funcionava no período noturno, e durante tempestades, ainda sim, muito eficiente para a época, pois conseguiam transmitir a mensagem por quilômetros.

Em 1837, nos Estados Unidos um pintor chamado Samuel Finley Breese Morse (Samuel Morse), desenvolveu o Telégrafo Elétrico, onde os usuários se comunicavam através de pulsos elétricos, interpretados utilizando o código Morse.

Linguagem de Programação:

Ao início do curso, vimos que o conceito da linguagem de programação, é um conjunto de regras sintáticas e semânticas que definem a estrutura de programas desenvolvidos em computadores; Uma linguagem formal que permite que os desenvolvedores construam instruções precisas para um computador executar determinadas tarefas. Essas linguagens são dividida em dois níveis: baixo e alto.

Baixo Nível: Linguagem mais próxima de máquinas, utilizando pulsos elétricos, para efetuar a comunicação entre os hardwares, como exemplo, os números binários, onde o "0” e "1” representam os pulsos.

Alto Nível: Está mais próxima da linguagem "humana”, a qual os desenvolvedores utilizam IDEs e Editores de Código-Fonte com linguagens intuitivas, para produzirem seus projetos.

O que é IDE ? ARRUMAR TEXTO !!!!!!!!!!!

IDE, é um "Ambiente de Desenvolvimento Integrado" (Integrated Development Environment), uma ferramenta para editar códigos, repleto de funcionalidades como, ferramentas de preenchimento de código, plugins, e muitos outros recursos para facilitar o processo de criação de softwares, aplicativos, jogos e projetos mais complexos.

O que é Editor de Código ?

Assim como um IDE, os Editores de Código possuem muitas características similares, porém, um pouco mais limitado, onde o programador muitas vezes utiliza estes editores para a criação de projetos mais simples, que não necessitem de aplicações mais complexas, e para desenvolver sites, como este que você está navegando para conhecer um pouco sobre nosso curso de ADS.

Agora que vimos algumas IDEs e Editores de Texto, vou apresentar a você o Visual Studio Code, o primeiro Software que utilizamos em nosso curso para desenvolver os projetos iniciais.

O Visual Studio Code é um editor de código-fonte, executado em sua área de trabalho, com este Software podemos desenvolver sites e alguns aplicativos. Ele oferece suporte a extenções para facilitar ainda mais o seu trabalho.
Com ele, realizamos nossos primeiros projetos utilizando as linguagens de "html", "css", e "JavaScript"

  • HTML

    • Conhecido como "Hypertext Markup Language" (linguagem de marcação de Hipertexto), é utilizada construir conteúdos em sites utlizando suas < tags >, com ela, implementamos textos, imagens, vídeos, tabelas, e outros.

  • CSS

    • Conhecido como "Cascading Style Sheets" (Folhas de Estilo em Cascata), o css é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou em XML.

  • JavaScript

    • JavaScript é uma linguagem de programação interpretada e estruturada, possui script em alto nível com tipagem dinâmica fraca. criada para ser executada em navegadores ou aplicativos mobile, manipulando o comportamento de seus elementos, tornando-os mais dinâmicos. Pode ser usado também no back-end.

Oque são as Tags do Html ?

  • Tags

    • As tags HTML são elementos utilizados para marcar e formatar os conteúdos textuais ou visuais em documentos web. Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.
    As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que” ("<"), em seguida vem o nome do elemento, logo, “maior que” (">"),incerimos o conjunto de caracteres que serão agrupados com esta tag e para fecharmos o comando, abrimos novamente incerimos a "/" e fechamos.
    Nesta imagem a sua esquerda, você verá alguns exemplos de tags semânticas, utilizadas para organizar o corpo da página web.
    Outros exemplos de tags são :

    Tag Função
    <i> deixa o texto em itálico
    <u> deixa o texto sublinhado
    <strong> deixa o texto em negrito
    <sup> coloca um texto sobrescrito
    <mark> destaca um trecho de texto

CSS

É uma linguagem de estilização, sua função é descrever como os elementos de uma página web ou aplicativo mobile devem ser exibidos, configurando as posições dos elementos, cores, tamanhos e formas.
A sintaxe do CSS é bem simples, composta do seletor e o bloco de declarações.

  • Seletor

    • Identificador que define o que será considerado ou editado, do código HTML.
    Este seletor referencia "tags", "classes" e "id". As classes referenciadas por "ponto" ( . ), e os id por "hashtag" ( # )

  • Bloco de declarações

    • Este é o espaço onde estarão descritas as características do elemento que será estilizado dentre as "chaves" ( {...} ), e separados por "ponto e vírgula" ( ; ). Essas características serão descritas da seguinte forma:

      seletor{
    • propriedade: valor;
    • }

JavaScript

Quando uma página da web é carregada, o navegador cria um "Document Object Model" (DOM), que é um modelo de objeto de documento, definindo:

- Os elementos HTML como objetos.
- As propriedades de todos os elementos HTML.
- Os métodos para acessar todos os elementos HTML.
- Os eventos para todos os elementos HTML.

Em outras palavras, o DOM é um padrão dinâmico que descreve como obter, alterar, adicionar ou excluir elementos HTML utilizando o JavaScript.

OBS: passe o mouse sobre a imagem a sua esquerda para visualizar.
    Ao declararmos as variáveis, como por exemplo "var btn1" e "var btn2", podemos indicar qual elemento do HTML elas estão se referenciando utilizando o comando:

    var btn1 = document.getElementById("botao");

    • = - sinal de atributo
    • document - Referenciando o HTML
    • getElementById - Referenciando o objeto através do id.
    • getElementsByClassName - Referenciando o objeto através do nome da classe.
    • ("botao"); - É o id do objeto.
    OBS: caso você utilize a classe para referenciar, você deve selecionar qual objeto deseja , utilizando [0].

    Depois, adicionamos uma "lista de eventos" com a seguinte sintaxe:

    btn2.addEventListener("click",editar2);

    • btn1 é a variável
    • .addEventListener é o comando para adicionar uma lista de evento
    • "click" é a ação do mouse, que irá chamar a função.
    • editar1 é a função que este comando irá chamar.
